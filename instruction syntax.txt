DISPLAY [VALUE]
--------------------------------------------------------
ASSIGN [NAME] [TYPE] [VALUE]
--------------------------------------------------------
INPUT [TYPE] [NAME]
--------------------------------------------------------
IF [CONDITION]
	[INSTRUCTIONS]
ENDIF
--------------------------------------------------------
LOOP [CONDITION]
	[INSTRUCTIONS]
ENDLOOP
(the instructions are run while the condition is true)
--------------------------------------------------------
END



------------------------------------------------------------------------

\*[VALUE] is a string
\'[VALUE] is a number

conditions can only be one comparison, for example:
IF X = \'7
	DISPLAY \*Wrong answer
ENDIF

-----------------------------------------------------------------------------------------------
@sub
	[INSTRUCTIONS]
RETURN

declares a subroutine called sub, all subroutines are placed after the
END instruction
